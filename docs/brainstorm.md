# Key concepts and ideas 
The aim is to create a fun & nostalgic multiplayer collaborative "dungeon crawler" game, reminiscent of Dandy or Gauntlet with some modern twists.

## Main elements
* Multiplayer
* Collaborative
* Explore and solve
